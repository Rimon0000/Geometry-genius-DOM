Some Questions About DOM:

Question-1: What is DOM and what is the purpose of DOM?

Answer: DOM means Document Object Model. It is a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic.

Question-2: How will you select HTML elements using DOM? Name the DOM methods.

Answer: There are a five different methods for selecting an element in the HTML document.

  • 1. getElementById()
  • 2. getElementsByClassName()
  • 3. getElementsByTagName()
  • 4. querySelector()
  • 5. querySelectorAll()

Question-3: What is event bubble?

Answer: Event bubble is a method of event in the HTML DOM. It happens when an element receives an event, and that event bubbles up or propagated to its parent elements in the DOM tree until it gets to the root element.

Question-4: What is a callback function and why will you use it?

Answer: When a function call another function is called callback function. I use it because Callbacks make sure that a function is not going to run before a task is completed but will run right after the task has completed. It helps us develop asynchronous JavaScript code and keeps us safe from problems and errors.
